## HOW TO CONVERT HTML → IMAGE

1. Open the HTML file in Chrome
2. Right-click → Inspect → Toggle device toolbar (Ctrl+Shift+M)
3. Set dimensions to 1200 x 630
4. Take screenshot (Ctrl+Shift+P → type "screenshot" → "Capture screenshot")
5. Use that PNG for your social media post

**Or the quick way:** Open the HTML file → zoom your browser to fit → use any screenshot tool (Snipping Tool, CleanShot, etc.)
